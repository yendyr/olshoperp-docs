# Account Payment — Knowledge Base

## Fungsi menu

**Account Payment** mencatat pembayaran ke supplier (cash/bank + outstanding Purchase Invoice).

## Route

- Datalist / Create / Edit: `/accounting/supplier-payment`
- Create auto-POST → redirect edit

## Section

- Basic Information — Supplier (`Select supplier`)
- Account Payment Source — `Select Cash/Bank` → CashBank modal checkbox + Use
- Detail Account Payment — link **Outstanding Purchase Invoice** → modal + Use
